This Netlify site is only a landing page

FincaHub is built with Flask (Python). Netlify does not run that kind of web server on normal static hosting, so your deploy had no index.html and showed “Page not found.”

Host the real app on Render (or Railway, Fly.io, etc.): same GitHub repo, free tier available.

Deploy to Render

After deploy, open the URL Render gives you (for example https://ecotierra-fincahub.onrender.com). If your service name differs, replace it with yours.
